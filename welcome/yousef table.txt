ليوم الأول

الفطار: رغيف عيش بلدي + بيضة مسلوقة + خيار وطماطم.

سناك: تفاحة.

الغداء: صدر فرخة مشوي + ½ كوب رز + سلطة خضراء.

سناك: كوب زبادي لايت.

العشاء: تونة + جرجير + رغيف بلدي.


اليوم الثاني

الفطار: رغيف عيش بلدي + 2 معلقة جبنة قريش + خس وخيار.

سناك: برتقالة.

الغداء: سمك مشوي (200 جم) + 3 ملاعق مكرونة + سلطة.

سناك: حفنة لوز (10 حبات).

العشاء: علبة تونة + خس وخيار + نصف رغيف بلدي.


اليوم الثالث

الفطار: رغيف سن صغير + 2 معلقة فول بزيت زيتون خفيف + طماطم.

سناك: كمثرى.

الغداء: لحمة حمراء مسلوقة/مشوية (150 جم) + ½ كوب رز بني + خضار سوتيه.

سناك: كوب زبادي لايت + رشة شوفان.

العشاء: جبنة قريش + خيار + رغيف سن.

اليوم الرابع

الفطار: بيضة أومليت بزيت زيتون + شريحة توست أسمر + خضار.

سناك: تفاحة.

الغداء: سمك فيليه مشوي + بطاطس مسلوقة صغيرة + سلطة.

سناك: حفنة فول سوداني (غير مملح).

العشاء: تونة + خس + نصف رغيف بلدي.

ليوم الخامس

الفطار: رغيف بلدي + 2 معلقة جبنة بيضاء قليلة الدسم + خيار.

سناك: برتقالة.

الغداء: فرخة مسلوقة/مشوية (150 جم) + 3 ملاعق مكرونة + خضار مطبوخ.

سناك: كوب زبادي لايت.

العشاء: جبنة قريش + طماطم + رغيف سن.


اليوم السادس

الفطار: 2 شريحة توست أسمر + بيضة مسلوقة + جرجير.

سناك: كمثرى.

الغداء: لحمة مشوية (150 جم) + ½ كوب رز بني + سلطة خضراء.

سناك: حفنة جوز (3–4 حبات).

العشاء: تونة + خس وخيار + نصف رغيف بلدي.

اليوم السابع

الفطار: رغيف بلدي + 2 معلقة فول بزيت زيتون خفيف + طماطم.

سناك: تفاحة.

الغداء: سمك بلطي مشوي + بطاطس مسلوقة صغيرة + سلطة.

سناك: كوب زبادي لايت.

العشاء: جبنة قريش + خيار + رغيف سن.
